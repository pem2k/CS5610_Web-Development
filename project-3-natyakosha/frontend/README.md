# Natyakosha Frontend

This is the React + Vite frontend client side application for Natyakosha — The Dance Treasury.

## Tech Stack
* **Vite** for local development and HMR.
* **React** using state and effect hooks.
* **ESLint** flat config rules.
* **Prettier** code formatting.
* **Component-scoped CSS** stylesheets imported per component.

## Setup & Running
To run the frontend individually:
```bash
npm install
npm run dev
```
It will start the dev server at `http://localhost:5173/` and proxy `/api` calls to the local Express backend on `http://localhost:5000`.
